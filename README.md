# Breast-Cancer-Prediction-using-Machine-Learning

## 📌 Project Overview

This project focuses on Breast Cancer Prediction using Machine Learning algorithms.  
The objective is to classify tumors as:

- Benign
- Malignant

The project uses Logistic Regression and Support Vector Machine (SVM) models for prediction and classification.


## 📁 Dataset Information

The dataset contains medical diagnostic measurements used for breast cancer prediction.

- Input Features: Multiple medical attributes
- Output Classes:
  - Benign
  - Malignant
